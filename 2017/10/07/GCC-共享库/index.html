基本概念

共享库: 在程序执行期间需要调用到共享库的时候才加载到内存里面，可以被多个程序共享使用.

在 MacOS 上面创建和使用共享库要比在 Linux 上面简单.
在 Linux 上面还会牵扯到共享库路径问题, 在 Mac 上面就没有这样的问题.
